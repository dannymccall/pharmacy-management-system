# Pharmacy Management System

This project helps manage the sales, expenses, purchases, and profits of a pharmacy.

## Features
- Manage drugs, sales, and employees.
- Generate reports (daily, monthly, custom).
- Secure login and logout functionality.

## Setup
To get this project running in your system;

1. Donwload xampp and install
2. Copy the project folder into C:\xampp\htdocs Note: that is if you installed xampp in your C drive
3. Lunch xampp and then start the Apache and mysql server
4. Open your favourite browser search for http://localhost/phpmyadmin in your address bar
5. From the left pane you create a new database with the name "pharmacy"
6. From the top pane, you click on import and choose the "pharmacy.sql" in the project folder
7. Finally you open another tab in your favourite browser and search for http://localhost/dashboard and then you are good to go
